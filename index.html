APRENDA HTML DE FORMA FACÍL

Com este passo á passo,você sairá do zero e aprenderá HMTL.Vamos começar a jornada de codificação juntos!

Então,prepare-se para mergulhar no fascinate mundo do html, uma linguagem poderosa com muitos recursos. Depois de entender o básico, você pode aprender sobre tópicos mais avançados, como formulários html, tabelas html e muito mais. Lembre-se, a prática é a chave para se tornar proficiente em html!

  2. Incluindo títulos e parágrafos

  • Hello World

    • Um documento HTML básico tem uma estrtura específica, sempre começamos com a tag, >html>, >head> e bory>. a tag head é responsavél pelo título da barra do navegador;bory contém o conteúdo principa do documento HTML,que é exibido na janela principal do navegador,title define o título da página,que é exibido na barra de título do navegador,h1 esta tag o título é exibido como prioridadde ou seja bem grande.Além da tag incial que é html, que é importante para o inicio da página. Faça esse exemplo para a sua primeira codificação : > html> >head> >title> Titilo da Pagina >/title> >/head >bory >h1> WELLO,WORLD!>>h1> >/bory> >/html>. Agora que você tem a estrutura básica da sua página, você pode começar a adicionar conteúdo.

  • Incluindo títulos e parágrafos

    • Você pode combinar títulos e parágrafos para criar uma estrtura de conteúdo em sua página, para os títulos de cabeçalho existem 6 níveis, para você estilizar o seu texto do jeito que quiser, utilize h1 para um titulo grande e os demais h para diminuir o título, podendo usar até h6.Para pular linhas usamos >p>.

  • Adicionando links

    • Links são criados usando a tag de âncora >a>. A tag >a> deve conter um atributo href que aponta para o URL do link.Exemplo : > a heref ="https: //www.exemplo.com">. Neste exemplo o site citado é o URL para o qual link irá redirecionar.Existem outras possibilidade como Links Internos, ao usar >a> para criar links internos,ou seja,links que apontam para uma seção específica na mesma página .Para fazer isso você precisa utilizar o atributo id, junto com a letra >a>,Fazendo assim você criar link dentro do proprio site, direcionando o usuário em toquipos na mesma página.O outro modo de usar é abrir o link em uma nova guia,usando a > com href "mais o site" target= "_blank"e assim quando o usuário dar um click no link, vai abrir uma nova aba no navegador sem, fechar a aba em que esta o seu site.

  • Inserindo imagens

    • As imagens são inseridas usando a tag de imagem >img>.A tag >img> é uma tag vazia,o que significa que não tem uma tag de fechamento,Um dos atributos mais importante é src, é um caminho para a imagem que você deseja inserir.Outro atributo que achei pesquisando foi alt,fornece um texto alternativo para a imagem, caso ela não possa ser exibida. Isso pode acontecer se houve um erro no caminho do arviquivo, se a imagem não puder ser carregada.Além dos atribudos para moldar o tamanho da sua imagem que são width e height, sendo height a altura e width largura.

  • Estilizando o texto

    • A tag >strong> é usada para deixar as palavras em negrito, para itálico pode usar a tag >em>, e a tag >font> por sua vez ela da a vida ao seu site, dando a possibilidade de você pintar o texto.Usando a tag >u> para sublinhar o texto.
